RainyDays 🧥
Overview

RainyDays is a responsive e-commerce style frontend website built using HTML and CSS, designed to showcase a clothing brand’s landing page with categories, product listings, newsletter subscription, and about / contact sections. The theme is “rainy weather / jackets / outdoor apparel.”

The project is a course assignment for the Front-End Development (FED) course, implementing both desktop and mobile layouts (as per given design in Figma).

Features

Responsive layout (desktop + mobile)

Navigation bar with menu links (MEN, WOMEN, SALE, Search)

Hero / banner section with call-to-action (“SHOP NOW”)

Product categories listing (Men’s Jackets, Women’s Jackets, Sale)

“About Us” section describing brand story / tagline

Newsletter subscription form (email + subscribe button)

Products grid (“Popular Products”) with sample jacket items and size options

Footer / site sections: Products, Customer Service, About Us, Contact & Social Media info (as static links/icons)

Technologies Used

HTML5

CSS3 (Flexbox / Grid / standard layout techniques / responsive design)



How to Run / Use Locally

Clone the repository:

git clone https://github.com/denise-cramer/html-css-denisecramer.git


Navigate into project folder:

cd html-css-denisecramer


Open index.html in your web browser.

(Optional) Use a code editor (e.g. VS Code) to view/edit files.

No build tools or dependencies are required — this is a pure static site.
